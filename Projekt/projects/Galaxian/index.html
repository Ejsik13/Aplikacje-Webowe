GALAXIAN

PLAYER ONE

A - move left

D - move right

SPACE - shot

PLAYER TWO

ARROW LEFT - move left

ARROW RIGHT - move right

ARROW UP - shot

MENU

F2 - start the game

F3 - choose the number of players

F4 - change the number of players

F5 - reload the game
